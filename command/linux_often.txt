1. 系统指令
1.1 查看ip
ip addr

1.2 启动系统服务
systemctl start/restart docker
设置开机启动服务：
1.2.1 vi /etc/rc.d/rc.local,在最后行插入
systemctl start docker
docker start mongodb

1.2.2 修改权限
chmod +x rc.local


1.3 查看路径
pwd,pwdx 进程id

1.4 查看端口号
netstat -lnpt | grep 3306
安装netstat指令 yum install net-tools

1.5 修改服务器时间
data -s "2021-01-28 08:00:00"
写入bios
hwclock -w

1.6  重启、关机
reboot
halt


2. 用户与权限
2.1 文件权限 chmod 777 -R 文件名/目录
2.2 新增用户 useradd test
2.3 设置密码 passwd test
2.4 删除用户 userdel test
2.5 用户配置文件 /etc/passwd

3.1 .tar/.tar.gz

3.2 .gz
gunzip 文件名

3.3 .zip
unzip 文件名
zip 压缩文件名 目录

3.4 .rar
unrar 文件名
rar 压缩文件名 目录
